# Traffic-Light-Controller-using-Verilog
FSM-based Traffic Light Controller in Verilog with timing control using counters.

Traffic Light Controller designed using Verilog HDL based on a Finite State Machine (FSM).
Implements realistic signal timing using counters for highway and side roads.
Simulated and verified using waveform analysis in EDA Playground.
Demonstrates digital design concepts like state transitions, timing control, and synchronous logic.
